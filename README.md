# 🚀 Exercícios de Lógica: Estruturas de Repetição (do...while)

---

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** JavaScript (.js)
* **Ambiente:** Node.js / Browser Console
* **Editor:** VS Code
* **Controle de Versão:** Git & GitHub

---

## 📂 Desafios Desenvolvidos

* **1. Validação de Senha 🔐**
  - Implementação de um loop que solicita a senha ao usuário.
  - O acesso só é liberado ("Acesso Permitido") quando a senha fixa coincide com a digitada.
  
* **2. Soma com Confirmação ➕**
  - O programa acumula valores inteiros em uma variável de soma total.
  - Utiliza uma pergunta de confirmação (s/n) para decidir se continua ou para o laço.

* **3. Jogo de Adivinhação 🎲**
  - O usuário tenta adivinhar um número secreto entre 1 e 10.
  - Inclui um sistema de **contador** para informar ao final quantas tentativas foram necessárias.

* **4. Validação de Cadastro 📝**
  - Garante que nomes de produtos tenham no mínimo 3 caracteres.
  - Exibe mensagem de erro e repete a entrada caso a regra de negócio não seja atendida.

* **5. Sistema de Cafeteria (Desafio Final) ☕**
  - Menu interativo com opções de produtos e preços.
  - Soma os valores automaticamente e permite finalizar o pedido digitando "0".
  - Exibe o total da conta de forma clara ao encerrar.
